# PHP* Toolsuite

## What?

Contains scripts and configurations for:

* PHPCPD  
  bin/rebuy-phpcpd
* PHPCS  
  bin/rebuy-phpcs
* PHPMD  
  bin/rebuy-phpmd

## Installation

Add the rebuy composer repo and extend your require-dev section.

    "require-dev": {
        "rebuy/php-tools": "1.*"
    }
